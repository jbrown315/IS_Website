What is a Perceptron?

A Perceptron is the simplest form of an Artifical Neural Network, and the first type of ANN that was ever created.

Figure 1.

A Perceptron is an Artificial Neural Network that only has an input layer and an output layer, however no hidden layer.

Being a much simpler form of an ANN, Perceptrons are generally used to carry out "and/or" logic gate problems. "and/or" logic gates are a commonly used in all forms of programming. The logic gates are formatted as "x and y"/"x or y" where x and y are each statements which will return a true or a false. For "and", both statements must return true, for "or" at least one of the statements must return true.

How does a Perceptron work?

Figure 2.

A key piece of every Neural Network are the weights. In a Perceptron, the weights are randomly to each of the given inputs. When you run the code, the weights will be constantly adjusted through a method called Backpropagation.

Backpropagation simply compares the actual output to the expected output and then adjusts the weights accordingly.
